Canasta Familiar - Software de control de precios

Un carrito representa un conjunto de articulos con sus precios, para una determinada fecha y supermercado.
Este software permite llevar un registro de las variaciones de precio.

Usa una base de datos mysql, antes de comenzar, asegurarse de que la db 'canasta' existe:
create database canasta;

Luego al arrancar el software se encarga del resto.


Esta en fase experimental, y nacio como divertimento personal.
Por el momento no tengo intenciones de que esto cambie.
Al comienzo usaba una planilla de calculo, pero las planillas son aburridas!
