Canasta Familiar - Software de control de precios

Este software permite llevar un registro del precio de un artículo en un establecimiento, a una fecha determinada.

A partir de la información recogida, se pueden sacar reportes como por ejemplo:
+ Listado de precios de cada articulo, en todos los establecimientos.
+ Listado de compras efectuadas en un determinado período.

Usa una base de datos mysql.
Antes de comenzar, asegurarse de que la db 'canasta' exista:
$ mysql -e "create database canasta;"

Luego al arrancar el software se encarga de crear las tablas y demás.

Este software está en fase experimental.
Nació como divertimento personal, y por el momento no tengo intenciones de que esto cambie.
